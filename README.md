# 私人武器库

纯前端实现一些实用的一些小功能。

- 基于 [Vue](https://vuejs.org/) 的开发框架 [Nuxt.js](https://nuxtjs.org/)
- 使用 [BootstrapVue](https://bootstrap-vue.org/) 作为前端样式
- 引入 [Stylus](https://www.stylus.com/) 做为 CSS 扩展
